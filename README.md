# 🛡️ envman
![npm](https://img.shields.io/npm/v/@fronik/envman)
![license](https://img.shields.io/npm/l/@fronik/envman)

A secure CLI for managing environment variables with encryption, scanning, and diagnostics.

## 🚀 Installation
npm install -g @fronik/envman

## ⚡ Quick Start
envman init
envman scan

## 🔥 Features
- AES-256 encryption
- Secret detection (AWS, JWT, keys)
- Doctor command (health checks)
- Auto backups
- Safe sync

## 🛠️ Commands
envman init
envman add KEY=value
envman scan
envman doctor
